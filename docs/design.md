# Achtung die Kurva — Design Document

Date: 2026-05-07

## Overview

Локальная браузерная игра для 2–4 игроков в стиле Achtung die Kurve. Каждый игрок управляет «змейкой», которая постоянно движется вперёд и оставляет след. Игроки могут поворачивать налево/направо двумя клавишами. Цель — не врезаться в стены, чужие и свои следы. На поле появляются бонусы.

Игра распространяется как набор статических файлов — открывается двойным кликом по `index.html`, никаких сборщиков и Node.js.

## Files

- `index.html` — два экрана (меню и игра), переключаются через CSS-класс `active`.
- `style.css` — flexbox-разметка «поле + боковая панель», оформление меню.
- `game.js` — вся логика, разделена комментариями-секциями:
  1. Состояние игры
  2. Меню и настройка клавиш (с `localStorage`)
  3. Игровой цикл (`requestAnimationFrame`)
  4. Логика игроков (движение, поворот, рисование, столкновения)
  5. Логика бонусов (спавн, подбор, эффекты)
  6. Боковая панель счёта и экран между раундами

## Players and Controls

- Поддерживается 2, 3 или 4 игрока.
- Цвета по умолчанию:
  - Игрок 1 — красный
  - Игрок 2 — синий
  - Игрок 3 — зелёный
  - Игрок 4 — жёлтый
- Клавиши настраиваются игроком в меню. У каждого игрока две клавиши: поворот влево, поворот вправо.
- Одна клавиша не может быть назначена двум игрокам — при конфликте кнопка «Старт» неактивна.
- Настройки клавиш сохраняются в `localStorage`, чтобы при перезагрузке страницы не настраивать заново.

## Field and Game Loop

- Поле — квадрат 700×700 пикселей (canvas).
- Игровой цикл — `requestAnimationFrame`, ~60 кадров/сек.
- Каждый кадр для каждого живого игрока:
  - если зажата клавиша поворота, угол меняется на фиксированную величину;
  - точка двигается вперёд по углу с текущей скоростью;
  - линия от старой позиции к новой рисуется на canvas (если игрок не в фазе «разрыва»).

### Player state

- `x`, `y` — позиция головы
- `angle` — угол направления (радианы)
- `baseSpeed`, `speed` — базовая и текущая скорость
- `baseThickness`, `thickness` — базовая и текущая толщина следа
- `color` — цвет
- `alive` — жив ли
- `score` — очки
- `effects` — словарь активных эффектов с таймерами
- `gapTimer`, `inGap` — состояние «разрыва в следе»
- `recentPositions` — список последних точек для иммунитета к собственному хвосту

### Collisions

- Перед рисованием проверяем цвет пикселя в новой точке: если там уже что-то нарисовано — игрок умирает.
- Если новая точка вышла за границы 700×700 — игрок умирает.
- Иммунитет: не проверяем столкновение с пикселями, нарисованными в последние ~10 пикселей пути этого же игрока.

### Trail gaps

- Случайно раз в 1–3 секунды след игрока на ~150 мс становится прозрачным (линия не рисуется). Через разрыв можно проехать.

## Round and Scoring

- Раунд начинается: все живы, спавнятся в случайных точках поля под случайным углом.
- Перед стартом — отсчёт «3, 2, 1, GO!».
- Когда умирает игрок, **все выжившие получают +1 очко**.
- Раунд заканчивается, когда остался один или ноль живых.
- Игра идёт до **(N−1) × 10** очков, где N — количество игроков.

### Между раундами

- Поле тускнеет, появляется надпись: «Раунд X. Победитель: ИгрокY (+1)».
- Игра ждёт нажатия **Enter**.
- После Enter — отсчёт «3, 2, 1, GO!» и новый раунд.

### Конец игры

- Когда кто-то набрал нужные очки, на месте поля — надпись «Победил Игрок X!» и таблица итогов.
- Кнопки: «Играть снова» (тот же состав, очки обнуляются) и «В меню».

## Bonuses

### Spawn rules

- Каждые 3–7 секунд (случайно) на поле появляется бонус.
- Спавнится в случайной свободной точке (не на существующем следе).
- На поле одновременно — максимум 3 бонуса.
- **Бонус лежит на поле, пока его не подберут** (без таймера исчезновения).

### Pickup

- Когда голова живого игрока касается бонуса (расстояние меньше радиуса бонуса) — эффект применяется и бонус удаляется.

### Effect duration and stacking

- Длительность всех эффектов — **5 секунд** (кроме «Clear all trails», который мгновенный).
- **Таймеры одинаковых эффектов складываются**: если подобрал Speed up при активном Speed up — у тебя становится остаток + 5 секунд.
- Разные эффекты работают одновременно.

### List of bonuses

| Код | Название | Цвет | На кого | Эффект |
|---|---|---|---|---|
| A | Speed up | зелёный | себе | скорость ×1.7 |
| B | Slow down | оранжевый | врагам | скорость ×0.5 |
| C | Thin line | голубой | себе | толщина следа ×0.5 |
| D | Thick line | фиолетовый | врагам | толщина следа ×2 |
| E | Clear all trails | белый | всем | весь холст очищается мгновенно (следы исчезают), эффекта длительности нет |
| F | Reverse controls | красный | врагам | поворот инвертируется (клавиша «влево» поворачивает направо) |

## UI

### Главное меню

- Заголовок: **Achtung die Kurva**
- Выбор количества игроков: кнопки 2 / 3 / 4.
- Список игроков (по выбранному количеству), для каждого:
  - Цветная плашка с именем
  - Кнопка «Влево: <клавиша>»
  - Кнопка «Вправо: <клавиша>»
  - Клик по кнопке: показывает «Нажми клавишу…», ждёт нажатия, сохраняет.
  - Конфликт клавиш — кнопка подсвечивается красным.
- Кнопка «Старт» — активна, только когда все клавиши назначены и нет конфликтов.

### Игровой экран

Flex-разметка: слева — поле, справа — боковая панель.

- Слева: canvas 700×700.
- Справа (ширина ~250px), всегда видна:
  - Таблица счёта: цветные строки игроков, отсортированы по очкам по убыванию.
  - Под каждым игроком — иконки активных эффектов с таймерами (полоска или цифра).
  - Снизу — напоминание клавиш каждого игрока.
  - Маленькая кнопка «В меню».

## Out of Scope

- Сетевой мультиплеер.
- Боты / одиночная игра.
- Звук (можно добавить позже).
- Сохранение результатов между сессиями (только клавиши сохраняются).
- Мобильное управление (тач).
