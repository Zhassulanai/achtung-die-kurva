# Achtung die Kurva

Локальная браузерная игра для 2–4 игроков на одной клавиатуре в стиле Achtung die Kurve. Без сборщиков и Node.js — это статические `index.html` / `style.css` / `game.js`.

## Запуск

Открыть `index.html` двойным кликом или через любой статический сервер.

## Управление

Клавиши настраиваются в меню (по 2 на игрока — поворот влево/вправо) и сохраняются в `localStorage`.
